# Trasholini

A sustainable waste identification and disposal guidance application powered by AI.

## Overview

Trasholini is a cross-platform mobile application that helps users properly dispose of waste by using computer vision to identify items and provide personalized disposal instructions. The app combines Flutter on the frontend with a FastAPI backend to deliver real-time waste classification and eco-friendly disposal tips.

## Key Features

- **AI-Powered Waste Detection**: Scan waste items using your camera for instant identification
- **Smart Disposal Instructions**: Get personalized guidance on how to properly dispose of each item
- **Eco-Points System**: Track your environmental impact and earn points for responsible disposal
- **Disposal History**: View past scans and disposal records with images
- **Waste Categories**: Browse different waste types and disposal tips
- **Trash Bin Locator**: Find available disposal bins and their details
